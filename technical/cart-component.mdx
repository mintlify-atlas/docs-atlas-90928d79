## Overview

The shopping cart is a persistent component that:
- Stores items in browser localStorage
- Supports quantity adjustments
- Integrates with WhatsApp for order submission
- Handles products with custom measurements
- Toggles between pickup and delivery modes

## Cart State

Cart items are stored in an ES6 `Map`:

```javascript
// In-memory cart state
state.cart = new Map([
  ["cemento-25", 5],                    // Simple product
  ["viguetas-tensolite__3.00", 12],     // Product with meter selection
  ["hierro-8", 20]
]);
```

**Why Map?**
- O(1) lookups and updates
- Built-in iteration
- Clean API for additions/deletions
- Maintains insertion order

## Cart Rendering

### Render Function

```javascript app.js:577-623
function renderCart(refs, cartMap) {
  const items = Array.from(cartMap.entries())
    .map(([cartId, qty]) => {
      const product = resolveProductByCartId(cartId);
      if (!product) return null;
      return { ...product, cartId, qty };
    })
    .filter(Boolean);

  if (items.length === 0) {
    refs.cartItems.innerHTML = '<p class="cart-empty">Todavia no agregaste productos.</p>';
  } else {
    refs.cartItems.innerHTML = items
      .map(
        (item) => `
          <article class="cart-item" data-id="${item.cartId}">
            <div class="cart-item-header">
              <span>${item.name}</span>
              <span>${item.qty} ${item.unit}</span>
            </div>
            <div class="product-meta">Unidad: ${item.unit}</div>
            <div class="cart-item-controls">
              <button class="qty-btn" data-action="decrease" data-id="${item.cartId}" aria-label="Disminuir cantidad">-</button>
              <span class="qty-value">${item.qty}</span>
              <button class="qty-btn" data-action="increase" data-id="${item.cartId}" aria-label="Aumentar cantidad">+</button>
              <button class="btn btn-outline-dark" data-action="remove" data-id="${item.cartId}">Quitar</button>
            </div>
          </article>
        `
      )
      .join("");
  }

  const totalItems = items.reduce((sum, item) => sum + item.qty, 0);
  refs.cartTotal.textContent = `Pedido (${totalItems})`;

  if (items.length === 0) {
    refs.cartWhatsapp.classList.add("is-disabled");
    refs.cartWhatsapp.href = CART_WHATSAPP_BASE_URL;
  } else {
    const deliveryMode = getSelectedDeliveryMode(refs.deliveryToggleInput);
    refs.cartWhatsapp.classList.remove("is-disabled");
    refs.cartWhatsapp.href = `${CART_WHATSAPP_BASE_URL}?text=${encodeURIComponent(
      buildWhatsappMessage(items, totalItems, deliveryMode)
    )}`;
  }
}
```

## Cart Operations

### Add to Cart

```javascript app.js:930-934
function addToCart(cartMap, productId, qty, meterValue = "") {
  const cartId = buildCartItemId(productId, meterValue);
  const currentQty = cartMap.get(cartId) || 0;
  cartMap.set(cartId, currentQty + qty);
}
```

**Process:**
1. Build composite cart ID (handles meter selections)
2. Get current quantity (defaults to 0)
3. Add new quantity to existing
4. Update Map

**Example:**
```javascript
// Adding regular product
addToCart(state.cart, "cemento-25", 5, "");
// Cart: Map { "cemento-25" => 5 }

// Adding same product again
addToCart(state.cart, "cemento-25", 3, "");
// Cart: Map { "cemento-25" => 8 }

// Adding product with meter selection
addToCart(state.cart, "viguetas-tensolite", 12, "3.00");
// Cart: Map { "cemento-25" => 8, "viguetas-tensolite__3.00" => 12 }
```

### Update Quantity

```javascript app.js:936-946
function updateCartQty(cartMap, cartId, delta) {
  const currentQty = cartMap.get(cartId);
  if (!currentQty) return;

  const nextQty = currentQty + delta;
  if (nextQty <= 0) {
    cartMap.delete(cartId);
  } else {
    cartMap.set(cartId, nextQty);
  }
}
```

**Delta-based updates:**
- `delta = 1`: Increase quantity
- `delta = -1`: Decrease quantity
- Auto-removes item if quantity reaches 0

### Remove from Cart

```javascript app.js:948-950
function removeFromCart(cartMap, cartId) {
  cartMap.delete(cartId);
}
```

Simple deletion from Map.

## Event Handlers

### Add from Product List

```javascript app.js:296-314
refs.productList.addEventListener("click", (event) => {
  const target = event.target;
  if (!(target instanceof HTMLElement)) return;
  if (target.dataset.action !== "add") return;

  const productId = target.dataset.id;
  if (!productId) return;

  const product = productIndex.get(productId);
  if (!product) return;

  const card = target.closest(".product-card");
  const qtyInput = card ? card.querySelector("[data-qty-input]") : null;
  const meterInput = card ? card.querySelector("[data-meter-select]") : null;
  const qty = parsePositiveInteger(qtyInput instanceof HTMLInputElement ? qtyInput.value : "1");
  const selectedMeter = meterInput instanceof HTMLSelectElement ? meterInput.value : "";
  addToCart(state.cart, productId, qty, selectedMeter);
  saveCartToStorage(state.cart);
  renderCart(refs, state.cart);
});
```

**Event delegation on product list:**
1. Click event bubbles up to product list container
2. Check if target has `data-action="add"`
3. Extract product ID from `data-id`
4. Find product card to read quantity input
5. Read meter selection if present
6. Add to cart
7. Save to localStorage
8. Re-render cart UI

### Cart Item Controls

```javascript app.js:316-330
refs.cartItems.addEventListener("click", (event) => {
  const target = event.target;
  if (!(target instanceof HTMLElement)) return;

  const action = target.dataset.action;
  const productId = target.dataset.id;
  if (!action || !productId) return;

  if (action === "increase") updateCartQty(state.cart, productId, 1);
  if (action === "decrease") updateCartQty(state.cart, productId, -1);
  if (action === "remove") removeFromCart(state.cart, productId);

  saveCartToStorage(state.cart);
  renderCart(refs, state.cart);
});
```

**Actions:**
- `data-action="increase"`: +1 quantity
- `data-action="decrease"`: -1 quantity (auto-removes at 0)
- `data-action="remove"`: Delete item immediately

## Composite Cart IDs

Products with custom measurements use composite IDs:

### Build Cart ID

```javascript app.js:1245-1249
function buildCartItemId(productId, meterValue) {
  const cleanMeter = String(meterValue || "").trim();
  if (!cleanMeter) return productId;
  return `${productId}__${cleanMeter}`;
}
```

**Examples:**
```javascript
buildCartItemId("cemento-25", "")         // "cemento-25"
buildCartItemId("viguetas-tensolite", "3.00")  // "viguetas-tensolite__3.00"
```

### Resolve Product by Cart ID

```javascript app.js:1251-1267
function resolveProductByCartId(cartId) {
  const direct = productIndex.get(cartId);
  if (direct) return direct;

  const [baseId, meterValue] = String(cartId || "").split("__");
  if (!baseId || !meterValue) return null;

  const baseProduct = productIndex.get(baseId);
  if (!baseProduct || !Array.isArray(baseProduct.meterOptions)) return null;
  if (!baseProduct.meterOptions.includes(meterValue)) return null;

  return {
    ...baseProduct,
    id: cartId,
    name: `${baseProduct.name} ${meterValue}m`,
  };
}
```

**Resolution Logic:**
1. Try direct lookup (handles simple products)
2. Split on `__` separator
3. Validate base product exists
4. Verify product has `meterOptions`
5. Confirm meter value is valid option
6. Return synthetic product with modified name

**Example:**
```javascript
resolveProductByCartId("cemento-25")
// { id: "cemento-25", name: "Cemento x25kg", unit: "Bolsa" }

resolveProductByCartId("viguetas-tensolite__3.00")
// { id: "viguetas-tensolite__3.00", name: "Viguetas tensolite 3.00m", unit: "Unidad" }
```

## Quantity Parsing

```javascript app.js:952-955
function parsePositiveInteger(rawValue) {
  const parsed = Number.parseInt(String(rawValue || "").trim(), 10);
  return Number.isFinite(parsed) && parsed > 0 ? parsed : 1;
}
```

**Validation:**
- Trims whitespace
- Parses as base-10 integer
- Checks if finite number
- Ensures positive (> 0)
- Defaults to 1 on invalid input

**Examples:**
```javascript
parsePositiveInteger("5")      // 5
parsePositiveInteger("0")      // 1 (default)
parsePositiveInteger("-3")     // 1 (default)
parsePositiveInteger("abc")    // 1 (default)
parsePositiveInteger("  10  ") // 10
```

## WhatsApp Integration

### Build Message

```javascript app.js:957-964
function buildWhatsappMessage(items, totalItems, deliveryMode) {
  const deliveryText =
    deliveryMode === "envio" ? "con envio" : "con retiro en el local";
  const lines = [`Hola quiero averiguar estos productos ${deliveryText}:`];
  items.forEach((item) => lines.push(`- ${item.name}: ${item.qty} ${item.unit}`));
  lines.push(`Total de items: ${totalItems}`);
  return lines.join("\n");
}
```

```text
- Hierro del 8: 12 Barra
- Viguetas tensolite 3.00m: 8 Unidad
Total de items: 25
```

## Cart Workflow

<Steps>
  <Step title="User Browses Catalog">
    Selects category, searches products
  </Step>
  
  <Step title="Add to Cart">
    - Enters quantity
    - Selects meters (if applicable)
    - Clicks "Agregar"
  </Step>
  
  <Step title="Cart Updates">
    - Item added to Map
    - Saved to localStorage
    - Cart UI re-renders
    - Total updates
  </Step>
  
  <Step title="User Adjusts">
    - Increase/decrease quantities
    - Remove unwanted items
    - Toggle delivery mode
  </Step>
  
  <Step title="Submit Order">
    - Click WhatsApp button
    - Opens WhatsApp with pre-filled message
    - User sends to business
  </Step>
</Steps>
