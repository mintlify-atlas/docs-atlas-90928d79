## Overview

Éxodo is a **pure client-side web application** with no backend infrastructure. All functionality is implemented using vanilla JavaScript, HTML5, and CSS3, with data persistence handled through browser localStorage.

## Architecture Type

**Static Multi-Page Application (MPA)** with shared component logic

- No server-side rendering
- No API endpoints
- No database layer
- Pure client-side state management
- localStorage for data persistence

### Scroll Effects System

The platform implements performant scroll effects using `requestAnimationFrame` and `IntersectionObserver`:

```javascript app.js:178-208
function initScrollEffects() {
  const nav = document.querySelector(".nav");
  const hero = document.querySelector(".hero, .page-hero");
  const prefersReducedMotion = window.matchMedia("(prefers-reduced-motion: reduce)").matches;

  const revealObserver = buildRevealObserver(prefersReducedMotion);
  registerReveal(document.querySelectorAll("section, h1, h2, h3, h4"), revealObserver, prefersReducedMotion);

  const updateNavState = () => {
    if (nav) nav.classList.toggle("is-scrolled", window.scrollY > 10);
  };

  let ticking = false;
  const updateParallax = () => {
    if (!hero || prefersReducedMotion) return;
    const offset = Math.min(window.scrollY * 0.08, 40);
    hero.style.setProperty("--hero-parallax", `calc(50% + ${offset}px)`);
    ticking = false;
  };

  const onScroll = () => {
    updateNavState();
    if (ticking || !hero || prefersReducedMotion) return;
    ticking = true;
    window.requestAnimationFrame(updateParallax);
  };

  updateNavState();
  if (!prefersReducedMotion) updateParallax();
  window.addEventListener("scroll", onScroll, { passive: true });
}
```

**Key Features:**
- Throttled scroll handling with `requestAnimationFrame`
- Respects `prefers-reduced-motion` user preference
- CSS custom properties for parallax effect
- Passive event listeners for performance

### Product Index

A `Map` is built for O(1) product lookups:

```javascript app.js:122
const productIndex = buildProductIndex(productCategories);

// Implementation
function buildProductIndex(categories) {
  const index = new Map();
  categories.forEach((category) => {
    category.items.forEach((item) => index.set(item.id, item));
  });
  return index;
}
```

## Performance Considerations

1. **Lazy Initialization**: Components only initialize if their DOM elements exist
2. **Event Delegation**: Single event listeners on parent containers
3. **Passive Listeners**: Scroll events use `{ passive: true }`
4. **RAF Throttling**: Animation updates batched with `requestAnimationFrame`
5. **Intersection Observer**: Progressive reveal animations

## Browser Compatibility

Required features:
- ES6+ JavaScript (Map, Arrow functions, Template literals)
- CSS Grid and Flexbox
- CSS Custom Properties
- LocalStorage API
- IntersectionObserver API

**Target**: Modern evergreen browsers (Chrome, Firefox, Safari, Edge)
