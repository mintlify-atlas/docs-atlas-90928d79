## Overview

The project slider is a responsive carousel that displays completed construction projects. It features:

- **Infinite loop scrolling** with cloned slides
- **Responsive slide count** (1 mobile, 2 tablet, 3 desktop)
- **Autoplay** with 5-second intervals
- **Dot navigation** for direct access
- **Smooth animations** with CSS transforms
- **Manual controls** (prev/next buttons)

## Infinite Loop Implementation

### Clone Generation

The slider clones slides on both ends to create seamless infinite scrolling:

```javascript app.js:705-741
const build = () => {
  logicalIndex = (currentIndex - cloneCount + originals.length) % originals.length;
  slidesPerView = getSlidesPerView();
  cloneCount = Math.min(slidesPerView, originals.length);
  section.style.setProperty("--slides-per-view", String(slidesPerView));

  // Remove existing clones
  Array.from(track.querySelectorAll("[data-clone='true']")).forEach((clone) => clone.remove());

  // Clone slides for tail
  const headClones = originals
    .slice(-cloneCount)
    .map((slide) => {
      const clone = slide.cloneNode(true);
      clone.dataset.clone = "true";
      return clone;
    });
  
  // Clone slides for head
  const tailClones = originals
    .slice(0, cloneCount)
    .map((slide) => {
      const clone = slide.cloneNode(true);
      clone.dataset.clone = "true";
      return clone;
    });

  // Insert clones
  headClones.forEach((clone) => track.insertBefore(clone, track.firstChild));
  tailClones.forEach((clone) => track.appendChild(clone));

  // Set slide widths
  const gap = Number.parseFloat(getComputedStyle(track).columnGap || "0");
  const viewportWidth = viewport.getBoundingClientRect().width;
  const cardWidth = (viewportWidth - gap * (slidesPerView - 1)) / slidesPerView;
  Array.from(track.querySelectorAll(".project-card")).forEach((card) => {
    card.style.flex = `0 0 ${cardWidth}px`;
  });

  currentIndex = cloneCount + logicalIndex;
  renderDots();
  updateTrackPosition(false);
};
```

### Clone Layout Example

With 5 original slides and `cloneCount = 2`:

```
[Clone 4] [Clone 5] | [Slide 1] [Slide 2] [Slide 3] [Slide 4] [Slide 5] | [Clone 1] [Clone 2]
                    ^
                    Starting position (currentIndex = 2)
```

### Infinite Sync Logic

```javascript app.js:692-702
const syncInfinite = () => {
  if (currentIndex >= originals.length + cloneCount) {
    // Scrolled past end - jump to start
    currentIndex -= originals.length;
    updateTrackPosition(false);
  } else if (currentIndex < cloneCount) {
    // Scrolled before start - jump to end
    currentIndex += originals.length;
    updateTrackPosition(false);
  }
  logicalIndex = (currentIndex - cloneCount + originals.length) % originals.length;
  renderDots();
};
```

**Triggering:**
Sync runs on `transitionend` event after slide animation completes:

```javascript app.js:791
track.addEventListener("transitionend", syncInfinite);
```

### Example Flow

<Steps>
  <Step title="User Clicks Next">
    ```javascript
    currentIndex += 1; // 6 → 7
    updateTrackPosition(true); // Animate to slide 7 (Clone 2)
    ```
  </Step>
  
  <Step title="Transition Completes">
    `transitionend` event fires
  </Step>
  
  <Step title="Sync Detects Clone">
    ```javascript
    if (currentIndex >= 7) { // 7 >= 7 (5 + 2)
      currentIndex -= 5; // 7 → 2 (jump to real Slide 2)
      updateTrackPosition(false); // Instant jump (no animation)
    }
    ```
  </Step>
  
  <Step title="User Sees Seamless Loop">
    Visual position appears unchanged, but physically jumped to real slide
  </Step>
</Steps>

## Track Transform Updates

```javascript app.js:666-672
const getStepWidth = () => {
  const first = track.querySelector(".project-card");
  if (!first) return 0;
  const firstWidth = first.getBoundingClientRect().width;
  const gap = Number.parseFloat(getComputedStyle(track).columnGap || getComputedStyle(track).gap || "0");
  return firstWidth + gap;
};
```

```javascript app.js:683-689
const updateTrackPosition = (withTransition = true) => {
  track.style.transition = withTransition
    ? "transform 0.72s cubic-bezier(0.22, 0.61, 0.36, 1)"
    : "none";
  const step = getStepWidth();
  track.style.transform = `translate3d(${-currentIndex * step}px, 0, 0)`;
};
```

**Parameters:**
- `withTransition = true`: Smooth animation (user navigation)
- `withTransition = false`: Instant jump (infinite loop sync)

```javascript app.js:780-789
dotsWrap.addEventListener("click", (event) => {
  const target = event.target;
  if (!(target instanceof HTMLElement)) return;
  const dot = target.closest("[data-dot-index]");
  if (!(dot instanceof HTMLElement)) return;
  const index = Number.parseInt(dot.dataset.dotIndex || "", 10);
  if (!Number.isFinite(index)) return;
  goToLogical(index);
  startAutoplay();
});
```

## Autoplay System

### Start Autoplay

```javascript app.js:766-769
const startAutoplay = () => {
  stopAutoplay();

};
```

**Interval:** 5000ms (5 seconds)

### Stop Autoplay

```javascript app.js:760-764
const stopAutoplay = () => {
  if (autoplayTimer) window.clearInterval(autoplayTimer);
  autoplayTimer = 0;
};
```

### Pause on Interaction

```javascript app.js:792-795
slider.addEventListener("mouseenter", stopAutoplay);

slider.addEventListener("focusin", stopAutoplay);
slider.addEventListener("focusout", startAutoplay);
```

**Triggers:**
- Mouse enters slider → Pause
- Mouse leaves slider → Resume
- Focus enters (keyboard nav) → Pause
- Focus leaves → Resume

## Responsive Rebuild

```javascript app.js:796-799
window.addEventListener("resize", () => {
  window.clearTimeout(initProjectsSlider.resizeTimer);
  initProjectsSlider.resizeTimer = window.setTimeout(() => build(), 120);
});
```

**Debounced resize handler:**
1. Clear existing timeout
2. Set 120ms timeout
3. Rebuild slider with new viewport size
4. Recalculates `slidesPerView`, slide widths, clones

<Note>
  The rebuild is debounced to avoid excessive calculations during resize. Uses function property `initProjectsSlider.resizeTimer` to store timeout ID.
</Note>

## Common Issues

<AccordionGroup>
  <Accordion title="Infinite loop not working">
    **Check:**
    - `transitionend` event listener attached
    - Clone generation in `build()` function
    - `syncInfinite()` logic for boundary detection
  </Accordion>
  
  <Accordion title="Slides wrong size after resize">
    **Solution:**
    - Ensure `build()` recalculates slide widths
    - Check gap calculation from computed styles
    - Verify flex basis is applied to all cards
  </Accordion>
  
  <Accordion title="Autoplay doesn't pause">
    **Check:**
    - Mouse/focus event listeners attached to slider
    - `stopAutoplay()` clears interval correctly
    - `startAutoplay()` called on mouse leave/focus out
  </Accordion>
  
  <Accordion title="Dots out of sync">
    **Solution:**
    - Call `renderDots()` after every index change
    - Verify `logicalIndex` calculation in `syncInfinite()`
  </Accordion>
</AccordionGroup>
