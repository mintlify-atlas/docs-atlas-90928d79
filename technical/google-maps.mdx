## Overview

The Éxodo platform uses **Google Maps iframe embed** to display the showroom and office location on the landing page. This is a simple, no-code integration that requires no API key for basic functionality.

## Implementation

### HTML Embed Code

```html index.html:296-311
<div class="showroom-map">
  <iframe
    title="Mapa Showroom y Oficina Central"
    src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d2912.891684292582!2d-65.70773952537152!3d-26.94792209467429!2m3!1f0!2f0!3f0!3m2!1i1024!2i768!4f13.1!3m3!1m2!1s0x942225d72b0d9079%3A0xe4c9ad7c7c882e1f!2s%C3%89XODO%20CORRALON!5e1!3m2!1ses-419!2sar!4v1770498857809!5m2!1ses-419!2sar"
    width="600"
    height="450"
    style="border:0;"
    allowfullscreen=""
    loading="lazy"
    referrerpolicy="no-referrer-when-downgrade"
  ></iframe>
  <div class="showroom-address">
    <strong>Ruta 355 El Mollar</strong>
    <span class="muted">Showroom y Oficina Central</span>
  </div>
</div>
```

## Map Features

### Interactive Elements

Users can interact with the embedded map:

- **Zoom:** Scroll wheel or +/- buttons
- **Pan:** Click and drag
- **Fullscreen:** Click fullscreen icon (enabled by `allowfullscreen`)
- **View in Google Maps:** Click business name or "View larger map"

### Disabled Features

The basic embed does **not** include:

- Directions from user's location (requires full Google Maps)
- Street View integration
- Place reviews/ratings
- Custom markers or styling

<Note>
  For advanced features, consider upgrading to Google Maps JavaScript API with custom implementation.
</Note>

## Business Hours Status

The showroom section includes dynamic status badges:

```javascript app.js:249-252
function initBusinessStatus() {
  setBusinessStatus("corralon", isBusinessOpen(8, 30, 18, 0));
  setBusinessStatus("constructora", isBusinessOpen(8, 30, 21, 0));
}
```

**Status Logic:**

```javascript app.js:254-261
function isBusinessOpen(openHour, openMinute, closeHour, closeMinute) {
  const now = new Date();
  const localDate = new Date(now.toLocaleString("en-US", { timeZone: "America/Argentina/Buenos_Aires" }));
  const currentMinutes = localDate.getHours() * 60 + localDate.getMinutes();
  const opening = openHour * 60 + openMinute;
  const closing = closeHour * 60 + closeMinute;
  return currentMinutes >= opening && currentMinutes < closing;
}
```

**Business Hours:**

| Service | Hours | Timezone |
|---------|-------|----------|
| **Corralón** | 8:30 - 18:00 | America/Argentina/Buenos_Aires |
| **Construcción** | 8:30 - 21:00 | America/Argentina/Buenos_Aires |

## Performance Optimization

### Lazy Loading

```html
<iframe loading="lazy" ...>
```

The `loading="lazy"` attribute defers iframe loading until user scrolls near it:

- **Benefits:** Faster initial page load, saves bandwidth
- **Browser Support:** Modern browsers (Chrome, Firefox, Safari, Edge)
- **Fallback:** Older browsers load immediately (no harm)

### Connection Preload

Add to `<head>` for faster map loading:

```html
<link rel="preconnect" href="https://maps.googleapis.com">
<link rel="dns-prefetch" href="https://maps.googleapis.com">
```

<Warning>
  Currently **not implemented** in the codebase. Adding these would improve map load time.
</Warning>

## Alternative: Static Map Image

For better performance, consider static image with link:

```html
<a href="https://maps.google.com/?q=-26.947922,-65.707739" target="_blank">
  <img
    src="https://maps.googleapis.com/maps/api/staticmap?center=-26.947922,-65.707739&zoom=15&size=600x400&markers=color:red%7C-26.947922,-65.707739&key=YOUR_API_KEY"
    alt="Mapa de ubicación"
    width="600"
    height="400"
  />
</a>
```

**Benefits:**
- Faster load (single image vs. iframe)
- No external JavaScript
- Better for email/print

**Drawbacks:**
- Requires API key
- Not interactive
- Costs per request after free tier

## Privacy Considerations

### Google Tracking

Google Maps iframe may track:
- User IP address
- Browser/device info
- Interaction with map (zoom, pan, clicks)

### GDPR Compliance

Consider adding to privacy policy:

> "Este sitio utiliza Google Maps para mostrar nuestra ubicación. Google puede recopilar datos de uso según su Política de Privacidad."

### Cookie-Free Alternative

Use `https://www.google.com/maps/embed` (already implemented) instead of `https://maps.google.com` - sets fewer cookies.
