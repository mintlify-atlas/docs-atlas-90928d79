## Overview

The platform displays real-time open/closed status badges for both Éxodo Corralón and Piedra Angular Constructora. Status is calculated client-side using Argentina timezone and updates on page load.

<Note>
  Business hours status is **NOT live-updating**. It calculates once when the page loads. Users must refresh to see updated status.
</Note>

## Status Calculation

### isBusinessOpen Function

Time-based status check (app.js:254-261):

```javascript
function isBusinessOpen(openHour, openMinute, closeHour, closeMinute) {
  const now = new Date();
  const localDate = new Date(
    now.toLocaleString("en-US", { timeZone: "America/Argentina/Buenos_Aires" })
  );
  const currentMinutes = localDate.getHours() * 60 + localDate.getMinutes();
  const opening = openHour * 60 + openMinute;
  const closing = closeHour * 60 + closeMinute;
  return currentMinutes >= opening && currentMinutes < closing;
}
```

**Algorithm:**

<Steps>
  <Step title="Get Current Time">
    Fetch system time using `new Date()`
  </Step>
  
  <Step title="Convert to Argentina Time">
    Use `toLocaleString` with timezone "America/Argentina/Buenos_Aires"
  </Step>
  
  <Step title="Calculate Minutes Since Midnight">
    Convert current time to total minutes: `hours * 60 + minutes`
  </Step>
  
  <Step title="Calculate Business Window">
    Convert opening and closing times to total minutes
  </Step>
  
  <Step title="Compare Range">
    Check if current time falls within business hours (inclusive start, exclusive end)
  </Step>
</Steps>

### Timezone Handling

**Argentina Timezone:**
```javascript
const localDate = new Date(
  now.toLocaleString("en-US", { timeZone: "America/Argentina/Buenos_Aires" })
);
```

**Why This Matters:**
- Server/client may be in different timezones
- Business hours are Argentina-local times
- Ensures correct status regardless of user location

**Argentina Time Details:**
- Timezone: UTC-3 (no DST)
- Province: Tucumán
- City: El Mollar

### Edge Cases

**Boundary Times:**
```javascript
return currentMinutes >= opening && currentMinutes < closing;
```

- At 8:30 AM: **OPEN** (inclusive start)
- At 6:00 PM (Corralón): **CLOSED** (exclusive end)
- At 8:29 AM: **CLOSED** (before opening)
- At 6:01 PM (Corralón): **CLOSED** (after closing)

**Weekends:**
<Warning>
  The function does NOT check day of week. It only validates time ranges. Weekend handling is not implemented, so the status would show "open" on Saturdays/Sundays during business hours.
  
  To fix:
  ```javascript
  const dayOfWeek = localDate.getDay(); // 0=Sunday, 6=Saturday
  if (dayOfWeek === 0 || dayOfWeek === 6) return false;
  ```
</Warning>

### setBusinessStatus Function

Updates badge appearance (app.js:263-277):

```javascript
function setBusinessStatus(name, open) {
  const badge = document.querySelector(`[data-status="${name}"]`);
  if (!badge) return;

  const text = badge.querySelector(".status-text");
  const statusText = open ? "Abierto ahora" : "Cerrado";
  if (text) {
    text.textContent = statusText;
  } else {
    badge.textContent = statusText;
  }

  badge.classList.remove("status-open", "status-closed");
  badge.classList.add(open ? "status-open" : "status-closed");
}
```

## Limitations & Improvements

### Current Limitations

<AccordionGroup>
  <Accordion title="No Live Updates">
    Status only calculates once on page load. If a user leaves the page open, it won't update when business hours change.
    
    **Solution:** Use `setInterval` to recalculate every minute:
    ```javascript
    function initBusinessStatus() {
      const update = () => {
        setBusinessStatus("corralon", isBusinessOpen(8, 30, 18, 0));
        setBusinessStatus("constructora", isBusinessOpen(8, 30, 21, 0));
      };
      update();
      setInterval(update, 60000); // Every 60 seconds
    }
    ```
  </Accordion>
  
  <Accordion title="No Weekend Handling">
    The function doesn't check day of week. It would show "open" on Saturdays/Sundays during business hour times.
    
    **Solution:** Add day-of-week validation:
    ```javascript
    const dayOfWeek = localDate.getDay();
    if (dayOfWeek === 0 || dayOfWeek === 6) return false;
    ```
  </Accordion>
  
  <Accordion title="No Holiday Handling">
    No awareness of Argentine public holidays (e.g., Christmas, New Year, Independence Day).
    
    **Solution:** Maintain holiday list:
    ```javascript
    const holidays = new Set(["2026-01-01", "2026-12-25", ...]);
    if (holidays.has(formatDateIso(localDate))) return false;
    ```
  </Accordion>
  
  <Accordion title="Hardcoded Hours">
    Business hours are hardcoded in JavaScript. Changes require code edits.
    
    **Solution:** Load from configuration or CMS:
    ```javascript
    const businessHours = await fetch('/api/business-hours').then(r => r.json());
    setBusinessStatus("corralon", isBusinessOpen(...businessHours.corralon));
    ```
  </Accordion>
</AccordionGroup>

### Timezone Edge Cases

**DST Transitions:**
Argentina doesn't observe Daylight Saving Time, so no special handling needed.

**User Timezone:**
The calculation works correctly regardless of user's timezone:
- User in USA (UTC-5) views site at 10:00 AM local time
- Argentina time is 12:00 PM (UTC-3)
- Corralón shows as "Abierto" (opened at 8:30 AM Argentina time)

**Server-Side Rendering:**
If this were SSR, timezone calculation would happen on server using:
```javascript
const argentinaTime = new Date().toLocaleString('en-US', {
  timeZone: 'America/Argentina/Buenos_Aires'
});
```

## Implementation Example

Here's a complete implementation with improvements:

```javascript
function isBusinessOpen(openHour, openMinute, closeHour, closeMinute) {
  const now = new Date();
  const localDate = new Date(
    now.toLocaleString("en-US", { timeZone: "America/Argentina/Buenos_Aires" })
  );
  
  // Check if weekend
  const dayOfWeek = localDate.getDay();
  if (dayOfWeek === 0 || dayOfWeek === 6) return false;
  
  // Check if holiday
  const dateStr = formatDateIso(localDate);
  const holidays = new Set([
    "2026-01-01", "2026-03-24", "2026-04-02", "2026-04-03",
    "2026-05-01", "2026-05-25", "2026-06-20", "2026-07-09",
    "2026-08-17", "2026-10-12", "2026-12-08", "2026-12-25"
  ]);
  if (holidays.has(dateStr)) return false;
  
  // Check time range
  const currentMinutes = localDate.getHours() * 60 + localDate.getMinutes();
  const opening = openHour * 60 + openMinute;
  const closing = closeHour * 60 + closeMinute;
  return currentMinutes >= opening && currentMinutes < closing;
}

function initBusinessStatus() {
  const update = () => {
    setBusinessStatus("corralon", isBusinessOpen(8, 30, 18, 0));
    setBusinessStatus("constructora", isBusinessOpen(8, 30, 21, 0));
  };
  
  update(); // Initial calculation
  setInterval(update, 60000); // Update every minute
}
```
